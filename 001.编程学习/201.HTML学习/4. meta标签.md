```html
<!DOCTYPE html>
<html lang="zh">
<head>
    <meta charset="UTF-8">
    <title>4.meta标签</title>
    <meta name="keywords" content="html,css">
</head>
<body>
    <!-- 用于设置网页的元数据
            title：作为搜索结果的超链接的文字显示
            charset：指定网页字符集
            name：   指定数据名称
            content：指定数据内容
                keywords  表示网站的关键字，可指定多个关键字，关键字使用，隔开

                description   用于指定网站的描述，网站得到描述显示在搜索引擎的搜索结果中
    -->
   
</body>
</html>
```
