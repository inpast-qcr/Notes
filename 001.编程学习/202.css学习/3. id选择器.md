### 1. 格式

​	#id 属性值{
​				属性：值；
​	}

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>3. id选择器</title>
    <style type="text/css">
        #id001{
            border: 5px dashed blue;
            color:yellow;
            font-size:20px;
        }
        #id002{
            border: 1px solid yellow;
            color:blue;
            font-size:30px;
        }
    </style>
</head>
<body>
    <div id="id001">div标签1</div>
    <div>div标签2</div>
    <span id="id002">span标签1</span>
    <span>span标签2</span>
</body>
</html>
```
