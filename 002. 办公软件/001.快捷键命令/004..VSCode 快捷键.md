## 常用
| 描述                                          | 快捷键                         |
| --------------------------------------------- | ---------------------|
| 1. 生成html                                   | ！+ tab               |
| 2. 单行注释                                   | ctrl + /               |
| 3. 多行注释                                   | ctrl + shift + /       |
| 4. 显示/隐藏 侧边栏                           | ctrl + b                |
| 5. 打开默认浏览器（下载 open in browers扩展） | 下载 open in browers扩展alt+ b |
| 6. 打开浏览器                                 | alt + shift + b                |
| 7. 打开终端                                   | ctrl + `                       |
| 8. 创建多个编辑器                             | ctrl + \                       |
| 9. 打开命令行                                 | ctrl + shift + p               |
| 10. 打开新窗口                                | ctrl + shift + n               |
| 11. 代码上移                                  | alt + ⬆                        |
| 12. 代码向上复制                              | alt + shift + ⬆                |
| 13. 当前行上方加1行                           | ctrl + shift + enter           |
| 14. 当前行下方加1行                           | shift + enter                  |
| 15. 删除整行                                  | ctrl + shift + k               |
| 16. 代码格式化                                | alt + shift + f                |
| 17. 重命名（全局替换）                        | F2                             |
|                                               |                                |
|                                               |                                |
